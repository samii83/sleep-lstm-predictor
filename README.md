# Sleep Pattern Prediction using LSTM

Time-series modeling project to predict sleep efficiency based on sequential screen-time and behavioral data.

## Features

- LSTM-based sequential model
- Baseline comparison: Linear Regression & Random Forest
- Evaluation using MAE and RMSE
- Feature scaling and preprocessing
- Model comparison pipeline

## Tech Stack

- PyTorch
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Structure

models/ → LSTM + baseline models  
training/ → training pipeline  
evaluation/ → metrics & comparisons
